# Primer Parcial — Informática General

**Artes Multimediales — UNA**

**Nombre/s y apellido/s:** Stepan Snygin

Sitio web estático en HTML y CSS sobre **Rafael Lozano-Hemmer**: biografía e introducción, obras destacadas y una sección sobre tecnología e interactividad.

## Contenido del proyecto

| Archivo / carpeta | Descripción |
|-------------------|-------------|
| `index.html` | Página de inicio |
| `obras.html` | Obras destacadas |
| `tecnologia.html` | Tecnología e interactividad |
| `css/styles.css` | Estilos compartidos |
| `assets/` | Recursos (imágenes, textos) |

## Cómo ver el sitio

Abrí `index.html` en el navegador (doble clic o arrastrar el archivo a una ventana del navegador). Las rutas relativas funcionan si mantenés la misma estructura de carpetas.

Si preferís un servidor local, desde esta carpeta podés usar por ejemplo:

```bash
python3 -m http.server 8080
```

Luego visitá `http://localhost:8080` en el navegador.
